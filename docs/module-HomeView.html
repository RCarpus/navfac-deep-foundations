Class: module:HomeView

module:HomeView()

new module:HomeView()

The home view is where logged in user's generally start. Several buttons are rendered that let the user choose what they want to do. The relative path for this view is "/home". Home view should only be accessible to logged in users, so the checkLoginStatus() inside componentDidMount() redirects the user to the WelcomeView "/" if the don't havea valide JWT.
